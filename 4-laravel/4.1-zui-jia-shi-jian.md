# 4.1 最佳实践

使用最佳实践可以帮助我们更好的维护代码，开发起来更加愉悦。

## 全局函数

在composer.json 中的 autoload 下增加

```php
"files": [
    "app/Helpers/functions.php"
]
```

```php
...
"autoload": {
    "classmap": [
        "database"
    ],
    "psr-4": {
        "App\\": "app/"
    },
    "files": [
        "app/helpers/functions.php"
    ]
},

...
```

然后执行：

> composer dump-auto
