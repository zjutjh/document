# 原创成绩API

### 地址
GET `http://api.jh.zjut.edu.cn/student/scoresZf.php`

### 参数
```
username=201312341234[学号]
password=xxxxx[密码]
year=2017[学年]
term=3[学期, 3为上学期，12为下学期, 16为短学期]
```
